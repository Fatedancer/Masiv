# Comic Viewer App

Esta es una aplicación web de visualización de cómics que utiliza Vue.js, Vuex y TypeScript.

## Configuración del Proyecto

Asegúrate de seguir estos pasos para configurar y ejecutar la aplicación en tu entorno local.

### Instalación de Dependencias

Antes de comenzar, asegúrate de tener Node.js y npm (Node Package Manager) instalados en tu sistema. Puedes descargarlos desde [nodejs.org](https://nodejs.org/).

Luego, abre una terminal y navega hasta el directorio raíz de tu proyecto. Ejecuta el siguiente comando para instalar las dependencias del proyecto:

```bash```
npm install


### Ejecución del Servidor Proxy
La aplicación utiliza un servidor proxy para acceder a la API de XKCD. Asegúrate de ejecutar el servidor proxy antes de iniciar la aplicación. Abre una terminal y navega hasta la carpeta raíz del proyecto. Luego, ejecuta el siguiente comando:

```bash```
node proxy.js

### Compilación y Ejecución de la Aplicación

Una vez que el servidor proxy esté en funcionamiento, puedes compilar y ejecutar la aplicación. Ejecuta el siguiente comando:

```bash```
npm run serve

Esto compilará la aplicación y la ejecutará en un servidor local. Abre tu navegador y accede a http://localhost:8081 para ver la aplicación en funcionamiento.

### Uso de la Aplicación

La aplicación te permitirá ver cómics de XKCD y navegar entre ellos. También podrás ver los detalles de cada cómic.
